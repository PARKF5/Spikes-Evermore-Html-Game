# Spikes Evermore
<b>Spikes Evermore - A Simple HTML Rage Game</b>
<b>Open Source and willing to recruit new people for asset design and coding support</b>

<b>How To Play Spikes Evermore - it's a lot, so strap in<b>

First, go to the latest released version of Spikes evermore

Go to index.html and copy the text

then go to <a href="https://playcode.io/">playcode</a> click start coding

Then drag the Style.css file out of the src folder

After this, you can delete the src folder

Then paste the index.html file from GitHub into the index.html file in PlayCode

Then go to the style.css file on GitHub and copy the text 

Then paste the style.css into the style.css in PlayCode

Then click the tiny button to full screen, and you should be in
